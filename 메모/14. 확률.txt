- 평균 : 대표값
- 편차 : 평균에서 얼마나 떨어져있는지
- 분산 : 편차제곱의 평균. 평균을 기준으로 얼마나 퍼져있는지
- 표준편차 : 평균에서 평균적으로 얼마나 떨어져있는지

표준정규분포 : x => x-평균/표준편차

표준화/정규화 : 모든 feature의 스케일을 맞춰서 모델이 feature 간에 공정하게 학습하도록 하는 것이 목적입니다

표준화 하는이유 : feature 간 단위와 범위를 맞추어, 한쪽 feature에 치우친 영향력을 줄이기 위해서
표준정규화(Z-Score) 하는 이유 : "feature 간 단위를 맞추고, 데이터 분포를 평균 0, 표준편차 1로 맞춰서 균일하게 만들기 위함
