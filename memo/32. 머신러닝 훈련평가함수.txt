[sklearn 머신러닝 훈련/평가 함수]

cross_val_score, cross_val_predict : 교차검증용 함수. cv파라미터 O. 훈련(fit)+평가(score) 동시실행. 함수 하나만 쓰면됨. cv파라미터명시하지 않아도 디폴트로 5번의 교차검증 수행 

model.fit + model.score	: 단순검증용 함수. cv파라미터 X. fit은 훈련만, score는 평가만 실행 두 함수는 같이 써야함.

[머신러닝 모델을 구성할때 유의해야할 점]
- 회귀/분류 종류에 따라 사용하는 모델 다름 : 회귀 : ~Regressor / 분류 : ~Classifier
- train_test_split 때 stratify 적용 여부 : 분류일때만 적용
- kfold 클래스 : 회귀 - KFold / 분류 - StratifiedKFold

분류모델용 데이터를 회귀모델로 구성하는 것은 되지만(성능은 별도) 반대는 십중팔구 에러가 난다.

기본적으로 sklearn 머신러닝 모델은 gpu가상환경이어도 cpu를 사용한다. (gpu cuda는 딥러닝 프레임워크 전용)
